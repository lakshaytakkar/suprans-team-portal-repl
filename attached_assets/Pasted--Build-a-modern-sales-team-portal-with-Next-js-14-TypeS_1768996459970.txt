
Build a modern sales team portal with Next.js 14+, TypeScript, Tailwind CSS, and shadcn/ui components. Use a clean Vercel-inspired dark/light theme. Include a role switcher in the top navigation bar to toggle between "Sales Executive" and "Superadmin" views.

## Tech Stack
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Zustand for state management
- Sample data stored in JSON (no database needed)

## Theme
- Vercel-inspired minimal design
- Dark mode default with light mode toggle
- Clean typography, subtle borders, neutral colors
- Primary accent: Blue (#0066FF)

## Global Layout
- Collapsible sidebar navigation
- Top bar with: Role Switcher dropdown, Theme toggle, User avatar menu
- Main content area with consistent padding

---

## ROLE: SALES EXECUTIVE

### Pages & Features

#### 1. Dashboard (`/dashboard`)
Stats cards showing:
- Total Leads (assigned to me)
- Active Leads (in pipeline)
- Won This Month (count + value)
- Follow-ups Due Today

Quick actions:
- "Add New Lead" button
- "Today's Follow-ups" list (max 5)

#### 2. My Leads (`/leads`)
Table view with columns:
- Lead Name, Company, Phone, Email
- Service Interested, Value (INR)
- Stage (badge: New, Contacted, Qualified, Proposal, Negotiation, Won, Lost)
- Next Follow-up Date
- Actions (View, Edit, Quick Call Log)

Filters: Stage, Service, Date Range
Search: Name, Company, Phone
Bulk actions: Assign Stage, Schedule Follow-up

#### 3. Lead Detail (`/leads/[id]`)
Left panel:
- Contact info (name, company, phone, email, address)
- Lead source, assigned date
- Current stage with stage change dropdown
- Deal value input
- Service interested (multi-select)

Right panel - Tabs:
- **Activity Timeline**: All interactions chronologically
- **Notes**: Add/view notes
- **Tasks**: Related follow-up tasks
- **Documents**: Uploaded files list

Actions bar:
- Log Call (modal: duration, outcome, notes)
- Log Meeting (modal: date, attendees, notes)
- Send WhatsApp (opens template selector)
- Send Email (opens template selector)
- Schedule Follow-up (date picker + reminder)
- Mark Won (modal: final amount, payment mode)
- Mark Lost (modal: reason dropdown, competitor if any)

#### 4. Follow-ups (`/follow-ups`)
Calendar view (week/day toggle) + List view
Shows scheduled follow-ups with:
- Lead name, time, type (call/meeting/task)
- Quick complete checkbox
- Reschedule option

Overdue section at top (highlighted red)

#### 5. Pipeline (`/pipeline`)
Kanban board with columns:
- New | Contacted | Qualified | Proposal | Negotiation | Won | Lost

Cards show:
- Lead name, company
- Value, service
- Days in stage
- Next follow-up indicator

Drag-drop to change stage (auto-logs activity)

#### 6. My Performance (`/performance`)
Charts:
- Leads by stage (pie chart)
- Conversion funnel (bar chart)
- Monthly trend (line chart)

Stats:
- Conversion rate, Avg deal size, Avg time to close

## ROLE: SUPERADMIN

### Pages & Features

#### 1. Admin Dashboard (`/admin`)
Team overview stats:
- Total Leads (all), New Today, Won This Month
- Revenue This Month, Pipeline Value
- Top Performer card

Charts:
- Team performance comparison (bar)
- Revenue trend (line)
- Lead source distribution (pie)

#### 2. All Leads (`/admin/leads`)
Same as sales executive leads table but:
- Shows "Assigned To" column
- Filter by sales executive
- Bulk reassign action
- Export to CSV

#### 3. Team Management (`/admin/team`)
Table of sales executives:
- Name, Email, Phone
- Active Leads, Won This Month, Conversion Rate
- Status (Active/Inactive)

Actions: Add User, Edit, Deactivate

#### 4. Lead Assignment (`/admin/assignments`)
Unassigned leads queue
Round-robin auto-assign toggle
Manual assign with drag-drop or dropdown
Assignment rules configuration (by service, by region)

#### 5. Reports (`/admin/reports`)
Pre-built reports:
- Daily Activity Report
- Weekly Sales Summary
- Monthly Performance Report
- Lead Source Analysis
- Service-wise Revenue

Export options: PDF, CSV, Email

#### 6. Settings (`/admin/settings`)
- Lead stages configuration (add/edit/reorder)
- Services list management
- Lead sources management
- Email/WhatsApp templates
- Notification preferences

## Key User Flows

### Sales Executive - Daily Flow
1. Login → Dashboard shows today's follow-ups
2. Click follow-up → Opens lead detail
3. Make call → Log call with outcome
4. Update stage if needed
5. Schedule next follow-up
6. Repeat for all due follow-ups

### Sales Executive - New Lead Flow
1. Receive notification of new lead assignment
2. Open lead → Review details
3. Make first contact call
4. Log call → Update stage to "Contacted"
5. Schedule follow-up
6. Continue nurturing until Won/Lost

### Superadmin - Morning Review Flow
1. Login → Admin Dashboard
2. Check overnight leads in unassigned queue
3. Assign leads to sales team
4. Review team's yesterday performance
5. Check any escalated deals

### Lead Won Flow
1. Sales exec clicks "Mark Won" on lead
2. Modal: Enter final amount, payment mode, notes
3. Confirmation → Stage changes to Won
4. Activity logged automatically
5. Shows in performance metrics
